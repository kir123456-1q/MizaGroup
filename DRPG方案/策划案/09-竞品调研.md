# 09 竞品调研

> 调研目的：明确当前市场上风评较好的 DRPG 代表作、流行趋势，以及对本项目（《异世界迷宫最深部为目标》Godot DRPG）的设计参考。  
> 调研时间：2026-07-02（基于公开评测、社区讨论与媒体报道整理）

## 9.1 DRPG 品类现状（简述）

**DRPG**（Dungeon RPG / 迷宫 RPG）源于 **Wizardry** 一脉：第一人称、格点/网格探索、队伍回合制、回城整备循环。

当前市场特征：

- 品类仍 **小众**，但 **HD 复刻 + 近年新作** 带来回流玩家
- 风评好的作品共性：**探索有回报、系统有深度、难度可调或教程友好**
- 分化明显：正统王道线（Wizardry / 世界树） vs 日式进化线（剧情、破墙、大人数编成）

与本项目关系：原著曾于 2015 年与 **Wizardry Online / Schema** 官方联动，类型定位以 **王道 DRPG** 最为正统。

## 9.3 重点竞品详表

### S+ 级：核心参考

#### 《真·女神转生 DEEP STRANGE JOURNEY》（奇异之旅 Redux）

| 项 | 内容 |
|----|------|
| 开发商 | Atlus |
| 平台 | 3DS（2017/2018）；无官方 PC 移植 |
| 风评 | 系列内 **最成熟 DRPG 之一**；中文社区称「真女神 + 世界树迷宫融合体」 |
| 核心特色 | Demonica **Main App / SubApp** 成长；Forma 素材研发；仲魔交涉/合体/解析；**四档难度**；Sector 式章节迷宫 |
| 好评原因 | 探索与实验室整备双循环；难度分层完善；Redux 取消 SubApp 上限大幅改善 QoL |
| 常见批评 | 系统密度高、上手陡；原版 DS SubApp 格子限制（Redux 已缓解） |
| **本项目参考** | **核心参考** — 详见 [10-真女神转生奇异之旅拆解.md](./10-真女神转生奇异之旅拆解.md) |
| 重点借鉴 | 「表示」式能力解锁、回城研发、陷阱/遇敌调控、难度倍率表、明雷强敌、解析成长 |

### S 级：标杆作品

#### 《世界树的迷宫》三部曲 HD Remaster（世界樹の迷宮 I·II·III）

| 项 | 内容 |
|----|------|
| 开发商 | Atlus |
| 平台 | Switch、PC 等 |
| 风评 | 媒体约 **8.5/10**；中日社区常称 DRPG「金字塔」 |
| 核心特色 | 地图绘制（触笔/摇杆）、F.O.E 明雷强敌、职业编成、三层难度（PICNIC / BASIC / EXPERT） |
| 好评原因 | 探索成就感强；复刻版降低门槛；美术与 BGM 统一 |
| 常见批评 | 操作仍偏老派；Atlus「冷饭」争议（本作复刻改动较大，风评相对正面） |
| **本项目参考** | 层主题设计、明雷 Boss、回城循环、难度分层、层数=进度 |

#### Wizardry 系列 / Wizardry Variants Daphne

| 项 | 内容 |
|----|------|
| 代表 | 经典 Wizardry 1～8；**Wizardry Variants Daphne**（2024，手游） |
| 平台 | 多平台移植；Daphne：iOS / Android |
| 风评 | Daphne 外媒约 **80～85 分**；称「很 Wizardry」「沉浸感强」 |
| 核心特色 | 第一人称、回合、永久死亡感、种族/职业编成、深渊探索 |
| 好评原因 | 正统 DRPG 手感在移动端复活；战斗紧张、氛围到位 |
| 常见批评 | Daphne：**Gacha、联网依赖、氪金、Bug** 影响长线口碑 |
| **本项目参考** | 状态/技能/队伍 UI；资源紧与撤退；原著官方联动验证的类型方向 |

### A 级：风评突出（剧情 + 系统）

#### 《鲁弗兰的地下迷宫与魔女的旅团》（ルフラン）

| 项 | 内容 |
|----|------|
| 开发商 | 日本一（NIS） |
| 平台 | Switch、PS、PC 等 |
| 风评 | DRPG 圈内 **大好评**；机核等长评称赞剧情与迷宫设计 |
| 核心特色 | 最多 40 人「人形兵」、**破墙**、敌人明雷、黑暗童话叙事 |
| 好评原因 | 地上活泼叙事 vs 地下留白惆怅；迷宫经得起「破墙」折腾 |
| 常见批评 | 职业仅 8 种且差异不大；结魂书魔法使用率偏低；整体偏简单 |
| **本项目参考** | 地上/地下叙事节奏；探索自由度；多单位编成思路 |

#### 《加露露瓦的迷宫与月之少女》（Galleria）

| 项 | 内容 |
|----|------|
| 开发商 | 日本一（NIS） |
| 系列 | 鲁弗兰续作（Refrain → Galleria） |
| 发售 | 2023 |
| 风评 | Metacritic **普遍好评**；部分媒体称「近年最佳迷宫 crawl 之一」 |
| 核心特色 | 极深队伍 Build、复杂系统、40～50 小时流程、反转叙事 |
| 好评原因 | 系统满足感强；迷宫层设计用心；剧情冲击力强 |
| 常见批评 | **上手门槛极高**；教程弹窗过多；续作惊喜感弱于前作 |
| **本项目参考** | 复杂编成与信息分层（避免开局信息轰炸）；Boss 与剧情节点 |

| 类型 | 代表 | 玩家为何给好评 |

| **大众好评型** | 世界树 HD、恶魔迷宫 Extra | 难度可调、探索正反馈清晰、视听完成度高 |

## 9.5 2024～2026 市场趋势

| 趋势 | 说明 | 对本项目启示 |
|------|------|----------------|
| **复刻回流** | 世界树 HD 吸引大量 DRPG 新玩家 | 垂直切片应提供 **难度选项** 与清晰教程 |
| **剧情加权** | 纯刷迷宫不足以撑起口碑 | 地上城镇、女主线、守护者战需 **叙事节点** |
| **难度分层** | 有简单模式的作品 Steam 评价更稳 | 建议：普通 / 硬核（管理领域外惩罚） |
| **手游复活 IP** | Daphne 热度高但商业化争议大 | 单机买断 **不学 Gacha**；可学第一人称氛围 |
| **机制融合** | 明雷 Boss、破墙、转生、多职业 | 可降低纯暗雷枯燥感；「注视」怪物=明雷信息 |

## 9.8 竞品结论（写进策划案的结论句）

1. **类型选择正确**：王道 DRPG 与原著、Wizardry 联动一致；**奇异之旅** 提供系统与难度设计的成熟范式。  
2. **差异化抓手**：小说式「表示」UI、混乱/？？？、管理领域规则、百层联合国世界观——竞品较少成套呈现。  
3. **垂直切片应对标**：**奇异之旅**（App 成长、难度表、实验室循环）+ 世界树（探索）+ Wizardry（战斗 UI）+ 鲁弗兰（剧情节奏）  
4. **商业化**：单机买断优先；若未来考虑长线运营，可参考 Daphne 的氛围与难度，**避免** Gacha 与强制联网。  
5. **风险**：DRPG 小众 — 需 **难度分层 + 前 30 分钟教程** 降低弃坑率。
