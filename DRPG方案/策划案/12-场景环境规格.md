## 12.1 迷宫分区视觉（第一人称）

### A. 管理领域外（1 层教学区）

| 要素 | 原著描写 | 画面规格 | Godot 实现 |
|------|----------|----------|------------|
| 结构 | 石制回廊，不见天光 | 直道/转角/死路 4 向预渲染或低模模块 | `scenes/dungeon/view/corridor_wild.tscn` |
| 光源 | **石壁自发淡光**，仄暗 | 墙 emission 淡蓝绿 `#a8c4b8` @ 15%；无点光源 | Shader `wall_bioluminescent.gdshader` |
| 地面 | 污秽石板、异臭（用视觉暗示） | 石板 albedo 带泥渍 normal；角落积水 decal | 材质 `floor_wild_01` |
| 细节 | 风化祭坛、青苔、蜡烛残渣、兽皮、古箭 | 祭坛 = 可交互事件格 `event_shrine` | MeshInstance3D 或 2D 叠层 |
| 氛围 | 重复走廊焦虑感 | 同主题 3 种墙纹变体 + `wall_variant` 字段 | 配置表驱动换贴图 |
| 音效 | 兽吼、虫鸣、脚步回响 | 混响长、低频足音 | `AudioStreamPlayer3D` + Reverb |

### B. 管理领域内（正道）

| 要素 | 原著描写 | 画面规格 | 实现 |
|------|----------|----------|------|
| 地面 | 平整、**加工矿物铺装** | 规整石砖，接缝对齐网格 | `floor_safe_01`，与野外明显对比 |
| 照明 | **每隔一段距离人工照明** | 壁龛魔石灯，暖白 `#e8d4a8` | OmniLight3D 每 4 格一盏 |
| 功能 | 安全撤退、不可恶意 PvP | UI 罗盘变绿；小地图正道高亮 | `mod_safe_path` 模块 |
| 过渡 | 涡波「与此前回廊截然不同」 | 进入正道播放 1s 色调 lerp + BGM 切换 | `DungeonZoneController` |

## 12.2 弗茨亚茨（地上）

| 场所 | 原著描写 | 场景设计 | 资源策略 |
|------|----------|----------|----------|
| **迷宫出口** | 警备兵指引、联合国氛围 | 石拱门 + 魔石线地面纹路（发光沟槽） | 模块化广场 `town_gate.tscn` |
| **兑换所** | 像古董屋，古董破烂遍地 | 暖光室内，货架堆叠，店主 NPC | 内景单屏 + 菜单 UI |
| **廉价旅店** | 住一晚仍贵；存档点 | 木床、油灯、窄走廊 | 交互 `inn_save` |
| **图书馆** | 世界观/魔石线说明 | 书架背景 + 图鉴解锁界面 | 静态 CG 背景可复用 |
| **医院** | 缇亚住院 | 白帘、药草气味（文案） | 剧情专用背景 1 张 |
| **街道** | 贵族骑士国、消费极高 | 石路 + **魔石线**（青白沟槽）；水渠远景 optional | 见 §12.4 |

## 12.5 第一人称资源生产管线

### 方案 A — 2D 预渲染（切片推荐）

```
Blender 直角走廊块 → 烘焙 1024 贴图 → 每格 4 方向 + 转角
  → Godot Sprite3D（Billboard）或 SubViewport 全屏 Quad
```

| 资源 | 数量（1 主题） | 来源 |
|------|----------------|------|
| 直道/转角/门/楼梯 | 各 2 变体 × 4 向 = ~24 张 | 自研 Blender 或委托 |
| 墙/地/顶 tile | 各 1 套 PBR | [ambientCG](https://ambientcg.com/) 石质 CC0 + 手绘污渍 |

### 方案 B — 低模 3D 模块（二期）

- Godot `GridMap` 或 `MeshLibrary` 拼接隧道  
- 参考开源：[tiled_dungeon](https://github.com/marius-test/tiled_dungeon)（Godot 4）

### 小地图

- **逻辑层**：`MapTile.csv`  
- **渲染层**：`TileMapDual` 插件（双网格），`32×32` 墙/路/门/玩家  
- 来源：自绘或使用 [Kenney Tiny Dungeon](https://kenney.nl/assets/tiny-dungeon) CC0 再加工
