# 10 《真·女神转生 奇异之旅》核心参考拆解

> **对象版本**：《真·女神转生 DEEP STRANGE JOURNEY》（日版名）/ *Shin Megami Tensei: Strange Journey Redux*（2017 3DS，2018 欧美）  
> 中文常见译名：深渊奇异之旅、深邃奇妙之旅  
> **定位**：本项目 **核心玩法参考**（与 Wizardry、世界树迷宫并列，偏「系统成熟度」与「难度/QoL 分层」）  
> 社区共识：**真女神转生 + 世界树迷宫的融合体** — 第一人称迷宫、地图绘制、F.O.E 级强敌、实验室强化、仲魔交涉与合体。

---

## 10.1 游戏概览

| 项 | 内容 |
|----|------|
| 开发商 | Atlus |
| 类型 | 第一人称 DRPG + 仲魔收集 + 阵营叙事 |
| 舞台 | 南极「施瓦兹瓦尔特」（Schwarzwelt）— 吞噬地球的异空间迷宫 |
| 基地舰 | Red Sprite（红之精灵号）— 实验室、商店、合体、存档 |
| 关卡结构 | **Sector（扇区）** 多座迷宫，相当于章节式大地图 |
| Redux 追加 | 新角色 Alex、新迷宫 **Womb of Grief（悲伤之 womb）**、3 条新结局、语音、约 50 名新仲魔 |

### 核心循环

```
探索 Sector（踩格绘图 / 搜 Forma）
  → 遇敌 / 明雷强敌 / 陷阱格
  → 回合战斗（弱点 / Co-Op / 交涉）
  → 解析仲魔 → 获得 Demon Source
  → 回 Red Sprite：Irving 研发 App / 造装备 / 合体
  → 解锁新 Sector 或更深楼层
  →（主线）阵营锁定 → 多结局
```

## 10.2 探索与迷宫系统

### 10.2.1 地图与移动

| 系统 | 说明 |
|------|------|
| 第一人称格点移动 | 上下左右转向、前进，经典 DRPG |
| 自动地图（AUTOPILOT） | 踏过的格子自动记录；玩家可手绘标注 |
| Sector 分层 | 每扇区多层（如 Antlia 1F～B4F），楼梯/传送点连接 |
| 黑暗区域（Dark Zone） | 无 Visualizer 则不可见，需对应等级 Visualizer |
| 陷阱格 | 伤害地板、毒/睡眠地板等；SubApp 可减轻或免疫 |
| 传送点 | Guidance Angel 可显示目的地 |
| 隐藏层 | Floor Search（**仅 Redux**）揭示隐藏楼层 |

### 10.2.2 门与障碍（与 Main App 绑定）

| 障碍类型 | 对应 Main App | 设计意图 |
|----------|---------------|----------|
| 锁定门 Lock A～E | Unlock A～E | 进度门；Rare Forma 驱动解锁 |
| 隐藏门 Gate A～E | Gate Search A～E | 秘密房间、宝箱、捷径 |
| 相位障碍 Phase A～B | Phase Shifter A～B | 需特定 App 才能通过的空间 |
| 不可见敌人 Enemy A～C | Enemy Search A～C | 类似 F.O.E / 特殊仲魔，明雷化 |
| 默认 | Enemy Appearance（Proximity Alert） | 罗盘红蓝：敌人接近预警 |

### 10.2.3 Forma（素材）体系

迷宫内特有物质，分三类：

| 类型 | 颜色/稀有度 | 用途 |
|------|-------------|------|
| **Base Forma** | 橙 A / 银 B / 金 C | Irving 实验室制造 **剑/枪/防具/饰品** 与消耗品 |
| **Assist Forma** | 辅助配方 | 强化装备属性、特殊效果 |
| **Rare Forma** | 唯一 | 交给 Irving **解析后消失**，用于研发 Main App 或解锁 SubApp 购买 |

获取方式：战斗掉落、格子搜索（需 Forma Search A/B/C）、仲魔升级赠礼、EX Mission 奖励。

**设计要点**：Rare Forma 是 **「能力门钥匙」**，与主线/支线进度强绑定，形成「探索 → 带回实验室 → 迷宫能力升级」的正向循环。

## 10.5 战斗系统

### 10.5.1 基本结构

| 项 | 说明 |
|----|------|
| 回合制 | 主角 + 最多 3 仲魔（Redux 可 4 槽相关调整见版本） |
| 主角武器 | **剑**（偏 Str）/ **枪**（偏 Ma），实验室 Forma 制造 |
| 属性 | 火/冰/电/风/光/暗/物理/枪/万能等；弱点/抗性/吸收/反射 |
| 状态异常 | 毒、束缚、石化、魅惑等；Brave Heart 等可对策 |
| 先制/伏击 | Surpriser / Hyakutaro；难度影响概率 |

### 10.5.2 Demon Co-Op（仲魔协击）

- 当 **击中弱点**（Redux 还包括 **暴击**）时，与攻击者 **同阵营（Law 蓝 / Neutral 白 / Chaos 红）** 的队友追加 **万能属性** 追击伤害  
- SubApp 可强化：弹射、暴击触发、跨阵营参与  
- 被动 **Preparedness**：Co-Op 伤害 +25%  

**设计意义**：鼓励 **阵营编成一致性** 与 **弱点扫描（Analysis）**，不是纯数值碾压。

### 10.5.3 仲魔交涉（Negotiation）

| 要素 | 说明 |
|------|------|
| 触发 | 战斗中选择 Talk；满月默认禁止（Lunatic 例外） |
| 阵营 | Law/Chaos/Neutral；**相性**影响态度与成功率 |
| 流程 | 对话选项 → 可能索要道具/Macca/加入/给物后逃跑 |
| 仲魔图鉴 | 全仲魔标明出现地点、掉落、技能 |
| 黑暗仲魔 | 通常需合体获得，不可直接交涉 |

### 10.5.4 解析（Analysis）与 Demon Source

| 项 | 说明 |
|----|------|
| Analysis | 多次战斗同一仲魔提升解析等级 |
| Demon Source | 解析满后获得；合体时注入 **额外技能** |
| 限制 | 每种 Source 同时只能持有一个；用掉即消耗 |
| Password | 图鉴密码可再召唤（贵）；NG+ 可召高于主角等级的仲魔（密码仲魔除外） |

### 10.5.5 合体（Fusion）与图鉴（Compendium）

| 项 | 说明 |
|----|------|
| 二身合体 | 两仲魔合成新仲魔，继承部分技能 |
| 特殊合体 | 指定多只特定仲魔 |
| 事故 | 满月/新月概率↑；Fracastoro / Copernicus 可强制 0% / 100% |
| Compendium | 已合体仲魔可 **花 Macca 再召**；可 **覆盖存档** 保存等级/技能 |
| 等级限制 | 不能合体出 **高于主角等级** 的仲魔（NG+ 放宽） |

### 10.5.6 Magnetite 与召唤

- 携带仲魔消耗 **Magnetite**（魔石）；回舰补充  
- 与 Macca（钱）区分：Macca 买 App/装备/再召；Magnetite 维持队伍存在

## 10.7 EX Mission（支线任务）

| 类型 | 作用 |
|------|------|
| 主线 MM | 推进 Sector、解锁 App Forma |
| EX Mission | 额外剧情、Macca、装备、SubApp 解锁、特殊合体 |

## 10.8 阵营与多结局

| 阵营 | 颜色 | 影响 |
|------|------|------|
| Law | 蓝 | Co-Op 对象；交涉偏好；路线 Boss |

### 10.9.2 难度设计解读

| 设计手法 | 作用 |
|----------|------|
| **双轴伤害倍率** | 同时调敌我输出，Casual 极宽容，Impossible 极残酷 |
| **逃跑阶梯** | Casual 必逃；Expert 降逃；Impossible **禁逃** — 强制硬打 |
| **陷阱随难度缩放** | 探索惩罚与难度挂钩，不仅是战斗 |
| **交涉独立档位** | Casual 降低招募门槛，不影响战斗硬核玩家选 Expert |
| **NG+ Impossible** | 给老玩家终极挑战，不影响一周目剧情体验 |
| **March to Death 禁用** | 最高难取消「主角阵亡不 Game Over」保险 |

**对本项目建议映射**：

| SJ 档位 | 本项目建议名 | 用途 |
|---------|--------------|------|
| Casual | 叙事 / 初见 | 教程层、管理领域内 |
| Standard | 标准 | 垂直切片默认 |
| Expert | 硬核 | 管理领域外、低补给 |
| Impossible | 挑战（NG+） | 百层后期或成就模式 |

## 10.10 Redux 相对原 DS 的强化汇总

| 类别 | 原版 DS | Redux |
|------|---------|-------|

### 10.11.3 核心参考结论

《奇异之旅》的价值不在于「仲魔数量」，而在于：

1. **探索能力以 App 模块成长** — 与「表示」扩展同源  
2. **实验室是第二游戏** — 整备与迷宫等权重  
3. **难度表格式调节** — 敌我倍率、逃跑、陷阱、异常一条表搞定  
4. **Redux 的 QoL** — 被动天赋树替代 SubApp 上限；**咏唱代价**替代 Magnetite 紧张感  

5. **原著咏唱** — SJ 给「能力成长」，咏唱给「力量与牺牲」；二者在公会整备所汇合（见策划案 11）
