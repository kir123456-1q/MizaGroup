# 01 项目概述

## 1.1 项目基本信息

| 项 | 内容 |
|----|------|
| 暂定名 | 异世界迷宫最深部为目标（DRPG） |
| 类型 | DRPG — Dungeon RPG，王道迷宫探索 RPG |
| 平台 | PC（首期）；后续可评估 Steam Deck / 移动端 |
| 引擎 | Unity 6 / Unity 2022 LTS（见技术方案） |

| 参考联动 | 2015 年 Wizardry Online / Schema 官方联动 |

## 1.2 电梯演讲（30 秒）

玩家扮演被抛入百层迷宫的相川涡波，在「状态表示」与资源紧缺的规则下，以攻略游戏的心态一层层向下。回城整备、组建队伍、面对每十层的守护者，最终朝向据说能实现任何愿望的最深部——同时维系与女主角们脆弱而危险的羁绊。

## 1.3 目标受众

- 喜欢 Wizardry、Elminage、真女神转生迷宫的玩家  
- 异世界轻小说读者，接受硬核数值与 meta 叙事  
- 接受「慢节奏、高算计、重试」的单机 RPG 玩家  

## 1.4 设计支柱

以下五条为所有系统设计的裁决标准：

1. **迷宫即游戏攻略** — 状态面板、技能点、持有物品、怪物位阶是核心玩法语言，非纯剧情装饰。  
2. **资源紧** — MP 枯竭、解毒药分类型、管理领域内外规则差异，迫使玩家规划撤退点。  
3. **队伍即剧情** — 缇亚、拉丝缇娅拉、玛利亚等入队与关系影响编成与事件，非纯战力单位。  
4. **层数即章节** — 每 10 层守护者 = 结构 Boss + 世界观揭露节点。  
5. **地上是呼吸** — 兑换、医院、租房、祭典与迷宫探索交替，避免纯刷层疲劳。

## 1.5 与原著的改编边界

| 保留 | 可简化 | 暂不纳入垂直切片 |
|------|--------|------------------|
| 状态/技能/持有物品 UI | 百层全做 → 先做 1～10 层 | 第四卷起劳拉维亚大会篇 |
| 缇达/阿尔缇守护者战 | 五国全开放 → 先做弗茨亚茨+瓦尔德 | Web 版后期元叙事 |
| 技能「？？？」与混乱累积 | 全女主路线 → 先做缇亚+拉丝缇娅拉 | 全 16 卷剧情 |
| 圣诞祭/前祭氛围 | 多人 PvP 规则 → 先做 NPC 队伍遭遇 | MMO 式联机 |

## 1.7 成功标准（垂直切片）

- 玩家能在 30 分钟内理解「表示」与撤退逻辑  
- 单次下本 20～40 分钟有明确目标与成败感  
- 完成 1～5 层主线并触发缇达遭遇事件  
- 回城后可卖魔石、买药、分配技能点、触发至少 2 段剧情
