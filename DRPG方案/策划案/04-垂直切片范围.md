# 04 垂直切片范围

> **第一期目标**：可发布的玩法 Demo，验证 DRPG 循环与「表示」特色，**不做百层全内容**。

## 4.1 范围总览

| 类别 | 纳入 | 不纳入 |
|------|------|--------|
| 迷宫层 | 1～5 层完整；6～10 层各 1 条短通路 + 第十层入口事件 | 11 层及以下 |
| 城镇 | 弗茨亚茨（出口区、兑换所、旅店、图书馆）；瓦尔德提及 | 五国全开 |
| 角色 | 涡波、缇亚、拉丝缇娅拉（NPC/治疗）；警备兵、店主 | 玛利亚、斯诺、芙兰琉莱入队 |
| 战斗 | 位阶 1～5 普通怪；缇达遭遇战（1 阶段） | 阿尔缇正式战 |
| 剧情 | 召唤→逃出→结识缇亚→组队探索→缇达事件 | 圣诞祭正篇、第二卷全文 |

## 4.2 关卡与地图

### 第一层（教学）

- 管理领域外小段：遇狼、学会战斗与表示  
- 正道接入：安全撤退线  
- 怪物：锯齿甲虫、黑环萤、大蛤（中毒教学）

### 第二层～第三层

- 标准走廊 + 岔路  
- 引入：冰结魔法组合（洒水 + Freeze）  
- 遇敌率中等

### 第四层～第五层

- 环境变体（潮湿/狭窄）  
- 小队探索者 NPC 擦肩而过（无 PvP）  
- **第五层底**：缇达/阿尔缇剧情战事件格

### 第六～十层（简化）

- 仅连通楼梯与 1～2 个战斗格，用于「曾到达更深」的成就感  
- 第十层：火焰景观 + 阿尔缇对话（无全 Boss 战）

## 4.4 验收标准（Definition of Done）

### 玩法

- [ ] 网格移动、转向、开门、上下楼无穿模  
- [ ] 遇敌进入战斗，胜利获得经验与魔石  
- [ ] 升级需玩家确认；技能点分配生效  
- [ ] 持有物品存取；装备更换改变战斗数值  
- [ ] MP 耗尽时玩家能主动撤退回城  
- [ ] 城内贩卖魔石、购买解毒药与补给  

### 内容

- [ ] 涡波 + 缇亚双人编成  
- [ ] 至少 8 种普通敌人  
- [ ] 缇达遭遇战可完成或撤退  
- [ ] 表示 UI 四类：状态/技能/持有物品/注视怪物  

### 技术

- [ ] 存档/读档在旅店  
- [ ] 配置表驱动怪物与掉落（改表不改代码）  
- [ ] 1920×1080 窗口化可玩  

### 体验

- [ ] 新玩家 10 分钟内理解移动与战斗  
- [ ] 单次下本有明确「目标层」与撤退理由
