# 06 美术与音频需求

## 6.1 视觉风格

| 项 | 建议 |
|----|------|
| 整体 | 日系奇幻，偏文库插画风（鹈饲沙树系） |
| 迷宫 | 石制回廊，苔痕，魔石微光；勿过度写实 |
| UI | 简洁 RPG，「表示」带轻微科幻/系统感 |
| 色调 | 迷宫冷色；弗茨亚茨地上暖色；火焰层橙红 |

**风格板**：准备 3 张 mood board（迷宫走廊、城镇广场、战斗 UI）。

## 6.2 角色美术

| 素材 | 规格 | 切片数量 |
|------|------|----------|
| 全身立绘 | 2048×3072 PNG 透明 | 涡波、缇亚、拉丝缇娅拉 × 各 1 |
| 半身/头像 | 512×512 | 战斗 UI 用，每人 1 |
| 表情差分 | 立绘上层或独立 | 每人 3（通常/战斗/伤） |
| 缇亚断手差分 | 立绘变体 | 1 |

## 6.3 迷宫美术

| 素材 | 规格 | 说明 |
|------|------|------|
| 墙面/地面/天花板 | 512 或 1024 可平铺 | 第 1 层石制 1 套 |
| 门/楼梯/宝箱 | 256～512 | 通用交互物 |
| 第一人称走廊 | 预渲染图或实时 3 面墙 | 每主题 4～8 张转角变体 |
| 小地图图块 | 32×32 | 墙/路/门/楼梯/玩家 |

**实现建议（Unity）**：  
- 方案 A：2D 预渲染背景切换（省性能，经典 DRPG）  
- 方案 B：低模 3D 隧道 + 烘焙贴图（易扩展多层）

## 6.4 怪物美术

| 素材 | 规格 | 切片 |
|------|------|------|
| 战斗图（侧视） | 1024×1024 | 8 普怪 + 缇达 |
| 图标 | 128×128 | 图鉴 |
| Boss 特写 | 可选全屏层 | 缇达 |

## 6.5 UI 美术

| 界面 | 必备元素 |
|------|----------|
| 标题 | Logo、新游戏/继续/设置 |
| 迷宫 HUD | 迷你层数、方向罗盘、菜单键 |
| 表示 | 状态/技能/物品三页；半透明底 |
| 战斗 | 指令栏、前后排、日志、目标光标 |
| 编成 | _drag 或选择前后排 |
| 城镇 | 设施按钮或简地图 |
| 对话 | 立绘、姓名、文本框、选项 |

字体：**思源黑体** 或 **阿里巴巴普惠体**；数字用等宽（状态面板）。

## 6.6 特效

| 特效 | 用途 |
|------|------|
| 魔石掉落光 | 战斗结束 |
| Ice / Freeze 粒子 | 冰结魔法 |
| Dimension 扫描线 | 索敌魔法 |
| 升级/称号 | 「深翠之始」等 |

## 6.7 音频

| 类型 | 数量（切片） | 备注 |
|------|--------------|------|
| BGM | 4 | 迷宫、战斗、城镇、Boss |
| SE | 20+ | 脚步、门、攻击、魔法、UI |
| 语音 | 0（可选） | 关键句 5 条可后期加 |

| BGM | 场景 |
|-----|------|
| dungeon_01 | 1～5 层探索 |
| battle_normal | 普通战 |
| town_fratsiats | 弗茨亚茨 |
| boss_tiida | 缇达战 |

## 6.8 资产目录规范（Unity）

```
Assets/
  Art/
    Characters/{id}/
    Monsters/{id}/
    Dungeon/Floor01/
    UI/
  Audio/
    BGM/
    SE/
```

命名：`chr_aikawa_portrait.png`, `mon_blackfly_battle.png`, `bgm_dungeon_01.ogg`
