# 03 系统设计

> 每个系统按「目的 / 规则 / 界面 / 数据 / 边界 / 验收」六块描述。

---

## 3.1 迷宫探索（P0）

| 块 | 内容 |
|----|------|
| **目的** | 提供 DRPG 核心：格点移动、方向、遇敌、岔路与楼梯 |
| **规则** | 玩家面向北/东/南/西；前进一格；撞墙无效；门需钥匙或开关；楼梯切换层；`正道` 格不可遇恶意 PvP（管理领域内） |
| **界面** | 第一人称 3 面墙视图；可选小地图；层数/坐标显示 |
| **数据** | `DungeonFloor`, `Map`, `MapTile`, `Encounter` |
| **边界** | 战斗中不可移动；某些格需 `Dimension` 或道具才可见 |
| **验收** | ① 走满 10×10 测试图 ② 上下楼梯切换层 ③ 踩事件格触发对话 |

**管理领域（原著）**

| 区域 | 规则 |
|------|------|
| 管理领域外 | 可被其他探索者攻击；无正道保护 |
| 管理领域内（正道） | 安全休息；可向出口撤退 |
| 层主领域 | Boss 战专用空间 |

## 3.2 战斗系统（P0）

| 块 | 内容 |
|----|------|
| **目的** | 回合制资源交换：HP/MP/道具 vs 敌人 |
| **规则** | 先攻由敏捷或固定「我方先」；前卫承受近战；后卫可使用魔法/道具；指令：攻击/技能/道具/防御/逃跑；逃跑成功率受层数与敌人影响 |
| **界面** | 前后排角色立绘或头像；指令菜单；战斗日志；目标选择 |
| **数据** | `Enemy`, `Skill`, `Item`, `Formula`, `AIPattern` |
| **边界** | MP 不足技能灰显；混乱过高可能限制指令（后期） |
| **验收** | ① 2v2 战斗打完 ② MP 消耗正确 ③ 逃跑/全灭处理 |

**站位**

```
[前卫1] [前卫2]  ← 敌方 →
[后卫1] [后卫2]
```

## 3.3 成长系统（P0）

| 块 | 内容 |
|----|------|
| **目的** | 体现「升级菜单」「技能点分配」小说 UI |
| **规则** | 经验达阈值后 **手动确认升级**（非自动，对应小说）；每级得 skill_point；分配至剑术/冰结/次元等；素质 7.00 影响成长上限（涡波） |
| **界面** | 「状态」「技能」「技能点的分配」三页签 |
| **数据** | `Character`, `LevelExp`, `Skill` |
| **边界** | 战斗中不可分配；满级技能不可再投点 |
| **验收** | ① 经验满后弹升级 ② 投点后伤害/耗蓝变化 ③ 存档保留 |

**涡波初期技能（第一卷）**

| 技能 | 初期等级 | 备注 |
|------|----------|------|
| 剑术 | 1.01 | 物理 |
| 冰结魔法 | 2.00 | Ice, Freeze |
| 次元魔法 | 5.00 | Dimension 索敌 |
| ？？？ | ？ | 混乱↑时冷静，垂直切片可做被动 |

## 3.4 物品与经济（P0）

| 块 | 内容 |
|----|------|
| **目的** | 魔石贩卖、补给购置、装备更换 |
| **规则** | 「持有物品」无限存储（非装备）；装备栏影响战斗参数；魔石按位阶定价；解毒药分 **毒种类型**（如仅克毒蜂，5% 解其他） |
| **界面** | 持有物品列表；装备界面；商店买卖 |
| **数据** | `Item`, `Shop`, `DropTable` |
| **边界** | 尸体/活物不可存入（原著） |
| **验收** | ① 拾取进持有物品 ② 装备攻防变化 ③ 商店买卖扣款 |

## 3.5 表示 UI（P0 · 特色）

| 块 | 内容 |
|----|------|
| **目的** | 还原小说「视网膜上的表示」，形成差异化 |
| **规则** | 念出或菜单打开：状态/技能/魔法/持有物品/怪物（注视）；支持简中；可切换显示详细数值 |
| **界面** | 半透明面板，不挡全屏；怪物「注视」显示位阶与名称 |
| **数据** | 只读角色与敌人表 |
| **边界** | 战斗中打开表示消耗一回合（可选，偏硬核） |
| **验收** | ① 状态与配置一致 ② 注视未知怪显示 ??? ③ 日语/简中切换 |

## 3.6 地上城镇（P1）

| 块 | 内容 |
|----|------|
| **目的** | 回城整备与情报获取 |
| **规则** | 弗茨亚茨：高消费、兑换所、图书馆；瓦尔德：低消费、适合长住（第二卷）；设施以菜单或 2D 热点进入 |
| **界面** | 城镇地图 / 设施列表 |
| **数据** | `Town`, `Shop`, `NPC`, `Dialogue` |
| **边界** | 夜间部分设施关闭（可选） |
| **验收** | ① 兑换所卖魔石得铜币 ② 旅店存档 ③ 图书馆解锁图鉴条目 |

**垂直切片设施**

- 迷宫出口（警备兵指引）  
- 兑换所  
- 廉价旅店（存档）  
- 图书馆（图鉴/世界观）  
- 医院（缇亚住院剧情）

## 3.7 剧情与好感（P1）

| 块 | 内容 |
|----|------|
| **目的** | 队伍成员入队、事件分支、祭典 |
| **规则** | Galgame 式对话树 + flag；好感影响是否同行/技能支援；关键节点不可回滚（守护者战） |
| **界面** | ADV 立绘 + 选项；地图内气泡简对话 |
| **数据** | `Quest`, `Dialogue`, `StoryFlag` |
| **边界** | 迷宫内长剧情仅在安全格触发 |
| **验收** | ① 缇亚入队 flag ② 拉丝缇娅拉治疗事件 ③ 前祭邀请分支 |

---

## 3.8 守护者战（P2）

| 块 | 内容 |
|----|------|
| **目的** | 每 10 层章节 Boss，机制战 |
| **规则** | 缇达（20 层）：试炼、单挑规则；阿尔缇（10 层）：火属性、可谈条件；允许「拒绝战斗→逃跑」剧情战 |
| **界面** | 专用背景 + 多阶段血条 |
| **数据** | `Boss`, `BossPhase`, `Dialogue` |
| **边界** | 垂直切片可做「遭遇+战前对话」，完整多阶段后续迭代 |
| **验收** | ① 第五层触发缇达事件 ② 战败/撤退有剧情 ③ 缇亚断手 CG/立绘差分 |

## 3.9 系统优先级总表

| 优先级 | 系统 |
|--------|------|
| P0 | 迷宫移动、战斗、成长、物品、表示 UI |
| P1 | 城镇、存档、遇敌表、基础剧情 |
| P2 | 状态异常、守护者、好感 |
| P3 | 祭典小游戏、五国切换、大会 |
