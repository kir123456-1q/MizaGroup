## 3.1 迷宫探索（P0）

| 块 | 内容 |
|----|------|
| **目的** | DRPG 格点移动；**管理领域外/内**视觉与规则差异 |
| **规则** | 四向移动；撞墙无效；门/楼梯；`正道` 安全且可撤退；陷阱格（SJ 式伤害%）；`zone=Wild/Safe` |
| **界面** | 第一人称走廊（石壁自发光 vs 正道魔石灯）；罗盘；小地图（TileMapDual） |
| **数据** | `DungeonFloor`, `MapTile`（含 zone）, `Encounter`, `UpgradeModule` |
| **边界** | 战斗中不移动；`mod_dimension_map` / Dimension 扩大地图揭示 |
| **验收** | ① wild→safe 色调切换 ② 陷阱格受伤随难度 ③ 10×10 测试图通关 |

**管理领域（原著）**

| 区域 | 规则 | 视觉（见 12） |
|------|------|---------------|
| 管理领域外 | 无正道保护；高遇敌 | 污秽石板、生物光墙、祭坛残迹 |
| 管理领域内 | 安全休息、撤退 | 矿物铺装、人工照明 |
| 层主领域 | Boss 空间 | 第五层底/第十层炎景 |

## 3.2 战斗系统（P0）

| 块 | 内容 |
|----|------|
| **目的** | 回合资源交换 + **咏唱读条风险** + 弱点连携 |
| **规则** | 先攻；前后排；攻击/技能/道具/防御/逃跑/**咏唱**；咏唱消耗 1 回合后次回合增效；可被打断 |
| **界面** | 立绘/头像；指令栏；战斗日志；咏唱诗句字幕 |
| **数据** | `Enemy`, `Skill`, `Chant`, `Formula`, `EnemyAI` |
| **边界** | MP 不足灰显；混乱限制指令；Expert 禁逃（对齐 SJ） |
| **验收** | ① 2v2 打完 ② 风刻印降 MP ③ 打断付半代价 ④ 碎冰连携 |

**站位** — 同前（前卫 2 / 后卫 2）。

**难度四档** — 见 10 文档 10.9；表驱动 `GameConst.difficulty_*`。

## 3.3 咏唱与咒术（P0 · 特色）

| 块 | 内容 |
|----|------|
| **目的** | 还原原著「以牺牲换力量」；提供高风险高回报决策 |
| **规则** | 咒术 ≠ 普通魔法；装备刻印后可对 tagged 技能咏唱；代价类型：时间/次日回复/层内 MP 禁回/移动减速 |
| **界面** | 技能子菜单「咏唱」；读条屏诗句；状态页显示 active debts |
| **数据** | `Chant.csv`, `GameSession.chant_debts` |
| **边界** | 火属性深咏唱切片不开放；人生咏唱仅图鉴 |
| **验收** | 见 [11](./11-咏唱咒术与整备系统.md) §11.7 |

**原著锚点**：192 话 — 时间与次日体力/魔力；炎蛇 — 燃烧内心；升级魔法 — 以毒为代价的咒术。

## 3.4 整备中枢（P0 · SJ 融合）

| 块 | 内容 |
|----|------|
| **目的** | 回城「第二游戏」：研发探索模块、购买/装备刻印 |
| **规则** | 魔石 + 层进度解锁 `UpgradeModule`；公会 UI 研发；刻印 2 槽（Expert 1） |
| **界面** | `guild_workshop`：模块树 + 刻印槽 + 预览效果 |
| **数据** | `UpgradeModule.csv`, `Shop`, `GameSession.installed_*` |
| **边界** | 切片 6 个模块、2 个刻印 |
| **验收** | ① 中毒后研发毒鉴别 ② 魔石不足无法研发 ③ 研发后下本生效 |

**地点**：弗茨亚茨兑换所旁 **公会整备所**（Red Sprite 实验室映射）。
