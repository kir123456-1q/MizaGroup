## 技术决策摘要

| 项 | 选型 |
|----|------|
| 引擎 | **Unity 2022.3 LTS** 或 **Unity 6** |
| 语言 | C# |
| 架构 | 分层 + 事件总线；玩法数据表驱动 |
| 迷宫 | 格点 `GridDungeon` + 第一人称 2D/3D 视图 |
| 战斗 | 回合状态机 + 指令模式 |
| 配置 | CSV → ScriptableObject / JSON |
| 存档 | JSON 本地档；Steam 云可后期 |
| UI | uGUI + TextMeshPro |
| 输入 | 新 Input System（键鼠 + 手柄） |
