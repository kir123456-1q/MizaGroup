## 技术决策摘要

| 项 | 选型 |
|----|------|
| 引擎 | **Godot 4.4+** |
| 语言 | **GDScript** |
| 架构 | 分层 + EventBus；表驱动 |
| 迷宫 | `DungeonGrid` + 预渲染/3D 视图；**TileMapDual** 小地图 |
| 战斗 | 回合 FSM + **咏唱读条** + 弱点连携 |
| 整备 | 模块（SJ Main App）+ 咒术刻印（原著代价） |
| 配置 | CSV → `.tres` / 运行时 Dictionary |
| 存档 | JSON v2（含模块/刻印/债务） |
| UI | Control + Theme |
