# 04 数据与配置方案

## 4.1 数据流

```
策划案/samples/*.csv
        ↓  Editor 菜单「Import Config」
Assets/_Project/Data/Generated/*.asset  (ScriptableObject)
        ↓  Boot 场景加载
GameDatabase (内存 Dictionary 索引)
        ↓
Gameplay / UI 只读查询
```

运行时进度（队伍、背包、flag）→ `GameSession` → `SaveManager` → JSON 文件。

## 4.2 ScriptableObject 设计

每种配置表对应一类 SO 容器：

```csharp
[CreateAssetMenu(menuName = "Miza/EnemyDatabase")]
public class EnemyDatabase : ScriptableObject {
    public List<EnemyDefinition> enemies;
    // BuildIndex() → Dictionary<string, EnemyDefinition>
}

[Serializable]
public class EnemyDefinition {
    public string id;
    public string nameKey;
    public int rank;
    public int hp, atk, def, agi;
    // ...
}
```

**原则**：
- Definition = 只读配置  
- RuntimeInstance = 战斗中临时 HP、Buff 等  

```csharp
public class EnemyInstance {
    public EnemyDefinition Def;
    public int CurrentHp;
    public List<StatusEffect> Statuses;
}
```

## 4.3 CSV 导入器（Editor）

路径：`Assets/_Project/Scripts/Editor/CsvImporter/`

功能：
1. 选择 `策划案/samples/` 或 `Data/SourceCsv/`  
2. 解析 CSV（处理引号、逗号）  
3. 生成/更新 `Assets/_Project/Data/Generated/EnemyDatabase.asset`  
4. 校验：重复 id、悬空 drop_id、负数 HP  

菜单项：
- `Miza/Import All Configs`
- `Miza/Validate Configs`

可选：用 Python 预校验 CI：

```bash
py tools/validate_configs.py 策划案/samples/
```

## 4.4 配置与策划案同步

| 源（策划维护） | Unity 内（程序生成） |
|----------------|----------------------|
| `策划案/samples/*.csv` | 可复制到 `MizaDRPG/Data/SourceCsv/` |
| 扩展表在 `策划案/07` 定义字段 | 导入器版本与字段同步更新 |

建议 Git 跟踪 **CSV 源文件**，Generated SO 可跟踪或 `.gitignore` 后 CI 生成（团队约定即可）。

## 4.5 存档格式

文件：`%AppData%/MizaDRPG/save_slot_0.json`

```json
{
  "version": 1,
  "timestamp": "2026-07-02T12:00:00",
  "session": {
    "party": [
      { "chrId": "chr_aikawa", "level": 3, "exp": 120, "hp": 45, "mp": 30,
        "skillLevels": { "sk_sword": 2, "sk_dimension": 3 },
        "equipment": { "Weapon": "it_iron_sword" }
      }
    ],
    "inventory": {
      "items": [
        { "id": "it_meat", "count": 5 }
      ]
    },
    "storyFlags": ["met_lastiera", "tia_joined"],
    "dungeon": {
      "currentFloor": 3,
      "playerX": 5,
      "playerY": 8,
      "facing": "North"
    },
    "gold": 150
  }
}
```

| 字段 | 说明 |
|------|------|
| version | 存档迁移用 |
| party | 运行时角色实例 |
| inventory | 持有物品（非装备栏） |
| storyFlags | 字符串列表 |
| dungeon | 可空；在城内时为 null |

**迁移**：`ISaveMigrator` 按 version 升级旧档。

## 4.6 本地化

`Localization.csv`：

```
key,zh_CN,ja_JP
chr_aikawa_name,相川涡波,相川渦波
en_blackfly_name,黑环萤,黒環萤
```

UI 禁止硬编码中文（除 Debug）。

## 4.7 地图数据格式

**首期**：CSV `MapTile.csv`

```
map_id,x,y,tile_type,event_id,wall_variant
map_01_01,0,0,Wall,,1
map_01_01,1,0,Floor,,0
map_01_01,5,3,StairsDown,,0
```

**二期可选**： [Tiled Map Editor](https://www.mapeditor.org/) 导出 JSON，自定义 `TiledImporter` 转 `MapTile`。

## 4.8 GameDatabase 启动加载

`Boot` 场景 `GameBootstrap`：

```csharp
void Awake() {
    var db = Resources.LoadAll<ScriptableObject>("Data/Generated");
    GameDatabase.Initialize(db);
    SaveManager.Initialize();
    SceneManager.LoadScene("Title");
}
```

## 4.9 与「表示」UI 的数据绑定

| UI 页 | 数据源 |
|-------|--------|
| 状态 | `GameSession.Party[leader]` + 公式算最大 HP |
| 技能 | `SkillDatabase` + 角色 skillLevels |
| 持有物品 | `GameSession.Inventory` |
| 注视 | 当前 `EnemyInstance.Def` + 未鉴定显示 `???` |

## 4.10 配置表清单（程序实现顺序）

1. `GameConst` → `LevelExp` → `SkillEffect`  
2. `Character` → `Skill` → `Item`  
3. `Enemy` → `DropTable` → `EncounterPool`  
4. `DungeonFloor` → `MapTile`  
5. `Shop` → `Dialogue` → `Quest`  
6. `Localization`
