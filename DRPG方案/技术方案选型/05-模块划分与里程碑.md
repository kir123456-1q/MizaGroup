## 5.4 创建 Unity 工程（下一步操作）

在仓库根目录执行（需本机已装 Unity Hub）：

1. Unity Hub → New Project → **2D (Built-in)** 或 **3D URP**  
2. 项目路径：`D:\SoftWare\GitFork\MizaGroup\MizaDRPG`  
3. 按 `03-架构设计.md` 创建 `Assets/_Project/` 目录  
4. 将 `策划案/samples/*.csv` 复制到 `Assets/_Project/Data/SourceCsv/`  
5. 实现 `CFG-01` 导入器后执行 Import  

## 5.5 风险与对策

| 风险 | 对策 |
|------|------|
| 第一人称资产量大 | 切片只用 1 套石廊 + 4 方向模板 |
| 战斗平衡耗时 | 先用表驱动，后期单独平衡周 |
| 剧情量超预期 | 对话全 CSV，首期纯文本无语音 |
| 范围蔓延 | 严格对照 `策划案/04-垂直切片范围.md` |
