## 5.2 里程碑定义

### Milestone 0 — 可走路（第 2 周）

- [ ] `MizaDRPG` Godot 4.4 可运行  
- [ ] Boot → Title → 测试迷宫  
- [ ] 格点前进/转向/撞墙  
- [ ] 走廊贴图随朝向切换  

### Milestone 1 — 可战斗（第 4～5 周）

- [ ] CSV 导入 Enemy/Skill/Chant  
- [ ] 遇敌进入战斗 UI  
- [ ] 攻击/咏唱读条/逃跑/胜利  

### Milestone 2 — 可循环（第 6～8 周）

- [ ] 兑换所 + **公会研发 mod_poison_id**  
- [ ] 旅店存档；装备刻印  
- [ ] 表示 UI；冰结 + 洒水组合  

### Milestone 3 — 垂直切片（第 9～12 周）

- [ ] 1～5 层 + 正道/野外视觉区分  
- [ ] 缇亚入队、缇达遭遇  
- [ ] ST001～ST008；Windows 导出

## 5.4 创建 Godot 工程（下一步）

1. Godot 4.4 → New Project → `MizaDRPG/`，Renderer **Forward+**  
2. 按 `03-架构设计.md` 建目录；注册 Autoload  
3. 安装 TileMapDual 到 `addons/`  
4. 复制 `策划案/samples/*.csv` → `data/source_csv/`  
5. 实现 `CFG-01` 后 Import  

## 5.5 风险与对策

| 风险 | 对策 |
|------|------|
| 预渲染资产不足 | 切片仅 wild/safe 各 1 套 |
| 咏唱系统复杂 | 先 2 条刻印 + 固定代价 |
| 双网格学习成本 | 仅小地图使用；主视图不用 |
| 范围蔓延 | 对照 `04-垂直切片` + `11` 第六节 |
